# AIO UI Visualization (可視化・レポート)

このスキルは、分析結果をマーケティング担当者や経営層に「直感的に伝える」ための、美しく使いやすいインターフェースとレポートのデザイン指針を定義します。

## 1. ダッシュボードの主要要素

### A. スコア表示 (Primary KPI)
- **AIO Score Meter**: 0-100のゲージ。色分け（赤・黄・緑）により一目で現状を把握。
- **モデル別詳細**: Perplexity, SearchGPTなどのアイコンを並べ、それぞれの最適化度をレーダーチャートで表示。

### B. 改善アクションの可視化
- **Before/After 比較**: 修正前後のテキストを左右に並べ、AIの理解度がどう変わるかを視覚的に提示。
- **チェックリスト形式**: 完了した改善項目を「済」にする達成感のあるUI。

### C. 競合比較 (Competitor Analysis)
- **引用シェアチャート**: 特定の重要キーワードにおいて、自社と競合がどれくらいの割合で引用されているかを円グラフや積層棒グラフで表示。

## 2. レポート出力 (PDF/Export)

- **プロフェッショナルな外観**: 営業資料としてそのまま使えるよう、清潔感のあるフォントと余白を使用。
- **サマリーセクション**: 冒頭に「現在の課題、優先順位、期待される結果」を簡潔にまとめる。
- **グラフの埋め込み**: Pythonライブラリ（Plotly, Matplotlib）を活用し、インタラクティブかつ高品質な図版をレポートに含める。

## 3. テクノロジースタックへの適用

- **Streamlit**: プロトタイプ段階での迅速な構築。`st.metric`, `st.plotly_chart` を活用。
- **Vite/React (Vue)**: 長期的なSaaS版での高度なインタラクション。
- **CSS**: グラスモーフィズムやアニメーション（Framer Motion）を取り入れ、「未来のツール」であることを演出。
